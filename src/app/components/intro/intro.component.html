📌 ¿Qué es Proyecto Mapas?

Proyecto Mapas es una aplicación interactiva donde puedes explorar ubicaciones, buscar direcciones y guardar tus lugares favoritos.

📍 Explora ciudades, 🔍Busca direcciones y ⭐Guarda tus lugares favoritos de manera sencilla y rápida.

🚀 ¿Cómo usar esta aplicación?

  • 1-Explora el mapa: Mueve y acerca el mapa interactivo.
  • 2-Busca lugares: Encuentra direcciones y puntos de interés.
  • 3-Guarda favoritos: Guarda tus lugares preferidos para acceder rápidamente.
